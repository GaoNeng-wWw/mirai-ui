# Mirai ui

Mirai Ui是一个免费且开源的UI组件库，由`vue.js`与`tailwind.css`开发

## ✨ 功能呢嗝

- Vue 3
- Type 友好
- Tailwindcss
- Treeshake 友好

## 📦 安装

```bash
npm install @miraiui-org/vue-components
```

```bash
yarn add @miraiui-org/vue-components
```

```bash
pnpm add @miraiui-org/vue-components
```
